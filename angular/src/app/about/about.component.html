Bienvenue dans le Projet iWebcamProtect

Nous sommes en dernière année de nos études d’ingénieur à l’ISIMA, l’Institut Supérieur d’Informatique de Modélisation et de leurs Applications. En début d’année, nous avons eu l’opportunité de proposer un projet en lien avec nos filières respectives, la F4 pour Mathis et la F5 pour Paul.

Notre objectif était de concevoir un projet qui nous permettrait de manipuler des données de manière sécurisée et aisée, tout en intégrant des aspects d’intelligence artificielle. Nous cherchions à stimuler notre réflexion ainsi que notre sens de la conception et du développement.

Après mûre réflexion, nous avons décidé de créer un Proof of Concept (POC) pour un système de surveillance connecté. Ce système, équipé d’une webcam, permettrait la détection de mouvement avec reconnaissance des formes détectées. Nous avons imaginé ce dispositif comme une solution de surveillance pour les foyers, utile notamment en cas de départ en vacances.

En accompagnement de la webcam, nous avons développé une application Web nommée "iWebcamProtect". Cette plateforme offre aux utilisateurs la possibilité de suivre l’état de leurs caméras au fil du temps. L’application permet de créer un compte de manière sécurisée ou de se connecter avec un compte Google. Une fois connecté, l'utilisateur peut ajouter une caméra et consulter les dernières informations, lui assurant de rester informé sur l’état de son foyer.

Pour assurer la liaison entre l’application web, la caméra et la base de données, nous avons développé une API, dont nous détaillerons les fonctionnalités par la suite.

Description du Projet

Le but de ce projet est de créer un POC d’un système de vidéosurveillance permettant de surveiller l’intérieur d’un logement grâce à la détection de mouvement. En plus de la détection, une application Web permettra de visualiser les données récoltées.

L’application Web disposera également d’une interface de gestion complète et intuitive pour les administrateurs, leur permettant de gérer les utilisateurs et leurs permissions. Un utilisateur pourra ajouter et visualiser l’état d’une caméra, ainsi qu’accéder aux images prises par celle-ci.

Cette application sera compatible avec tous les navigateurs Web. Nous l’avons développée avec le framework ASP.NET Boilerplate, qui regroupe deux technologies :

  • ASP.NET Core avec C# pour la partie backend
  • Angular avec TypeScript pour la partie frontend

La connexion à l’application se fait en créant un compte ou en utilisant la fonctionnalité SSO avec un compte Google grâce à Google API. Le développement des applications a été réalisé sous Visual Studio 2022 pour le backend et Visual Studio Code pour l’interface utilisateur.
